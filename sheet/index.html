A Spreadsheet in 328 bytes of JavaScript

Main logic by Ondras Zarovi (source), inspired by Thomas Street, golfed down by me, subzey, p01 and rlauck. See the commit history.

Features:
  • Excel-like syntax (formulas start with "=")
  • Support for arbitrary expressions (=A1+B2)
  • Automatic localStorage persistence
